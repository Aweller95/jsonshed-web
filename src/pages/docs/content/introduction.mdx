<h2 id="info">Info</h2>

<Paragraph />

JSONSHED is a fake REST api for testing and prototyping, built in Node.JS and TypeScript.

Five HTTP request methods are supported: GET, POST, DELETE, PUT, PATCH.

<ExampleTitle>

Base url:

</ExampleTitle>

<Example>

```bash
https://api.jsonshed.com
```

</Example>

<Paragraph />
At the moment there are 3 available resources: Users, Todos and Posts.

<Paragraph size={6} />

<h2 id="pagination">Pagination</h2>

<Paragraph />

All paginated requests are returned with an object containing the following information:

<Paragraph />

| Keys        | Types     | Description           |
| ------------|:---------:| ---------------------:|
| page        | int       | current page          |
| limit       | int       | quantity per page     |
| total pages | int       | total amount of pages |
| results     | array     | results               |

<ExampleTitle size={6}>

Example Request

</ExampleTitle>

<Example>

```bash
https://api.jsonshed.com/todos?page=2&limit=2
```

</Example>
