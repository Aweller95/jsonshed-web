<h1 id="users">Users</h1>

<Paragraph size={6} />

<h2 id="users-schema">Schema</h2>

<Paragraph />

| Keys          | Types         | Description                                                                     |
| --------------|:-------------:| -------------------------------------------------------------------------------:|
| id            | string        | user id                                                                         |
| title         | string        | user title                                                                      |
| firstName     | string        | user first name                                                                 |
| lastName      | string        | user last name                                                                  |
| email         | string        | user email                                                                      |
| avatar        | string (url)  | user avatar image                                                               |
| address       | object        | user address with the keys 'street', 'city', 'zipCode' and country (all string) |
| jobTitle      | string        | user job title                                                                  |


<Paragraph size={6} />

<h2 id="create-user">Create a user</h2>

<Paragraph />

A user can be created through the <Pre>/users</Pre> endpoint by using the POST method. The <Pre>id</Pre> will be auto-generated.

<ExampleTitle>

Example Request

</ExampleTitle>

<Example>

```bash
https://api.jsonshed.com/users
```

</Example>

<ExampleTitle>

Example Body

</ExampleTitle>

<Example>

```json
{
  "firstName": "Jerry",
  "lastName": "Roob",
  "email": "Jerry_Stokes43@hotmail.com"
}
```

</Example>

<Paragraph />

<ExampleTitle>

Return

</ExampleTitle>

<Example>

```json
{
  "firstName": "Jerry",
  "lastName": "Roob",
  "email": "Jerry_Stokes43@hotmail.com",
  "id": "11"
}
```

</Example>

Note: The resource is not going to be created in the databse. Instead, it will be faked as if and returned.
