<h1 id="users">Users</h1>

<Paragraph size={6} />

## Schema

<Paragraph />

| Keys          | Types         | Description                                                                     |
| --------------|:-------------:| -------------------------------------------------------------------------------:|
| id            | string        | user id                                                                         |
| title         | string        | user title                                                                      |
| firstName     | string        | user first name                                                                 |
| lastName      | string        | user last name                                                                  |
| email         | string        | user email                                                                      |
| avatar        | string (url)  | user avatar image                                                               |
| address       | object        | user address with the keys 'street', 'city', 'zipCode' and country (all string) |
| jobTitle      | string        | user job title                                                                  |

<Paragraph size={6} />

<Paragraph />

<h2 id="filter-users">Filter users</h2>

<Paragraph />

Using query parameters will allow you to filter users by their keys.

<Paragraph />

Available parameters:

<Li>id</Li>
<Li>firstName</Li>
<Li>lastName</Li>
<Li>email</Li>
<Li>jobtitle</Li>

<ExampleTitle>

Example Request

</ExampleTitle>

<Example>

```bash
https://api.jsonshed.com/users?firstName=Amalia
```

</Example>
